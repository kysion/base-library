# Base Library

这是一个Go基础库项目，提供了各种实用工具和功能。

## 安装

```bash
go get github.com/kysion/base-library
```

## 特性

- 基础模型支持
- 实用工具函数
- 钩子系统
- 常量定义

## 自动化发布

本项目配置了GitHub Actions自动化发布流程。当代码被合并到主分支时，将自动创建新版本并发布。详情请参阅 [CI/CD说明文档](.github/README.md)。
